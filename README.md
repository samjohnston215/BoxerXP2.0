# 🥊 BoxerXP 2.0

A fully self-contained, mobile-first boxing training tracker. Open `index.html` in any browser — no server or build step needed.

## What's New vs BoxerXP v1

| Feature | v1 | v2 |
|---|---|---|
| Stats | 4 (STR/SPD/STA/TAC) | **5** (+ DEFENSE) |
| Nav Tabs | 4 | **5** (+ Achievements) |
| Achievements UI | Stub only | **Fully rendered with progress** |
| Weight Log | List only | **List + canvas sparkline chart** |
| Timer | Fixed 3min/1min | **User-editable work & rest durations** |
| Daily Modifier | Cosmetic banner | **Actually boosts XP earned that day** |
| Style Archetypes | 4 | **5** (+ IRON WALL for defense builds) |
| XP Formula | pow(1.15) | **pow(1.18) — steeper, more satisfying** |
| Tasks | 3 fixed | **3 dynamically generated by rank tier** |
| Combo Counter | ❌ | **✅ In-timer punch combo tally** |
| Session streak | Resets on timer only | **Persists cross-day reliably** |

## How to Use

1. Clone or download the repo
2. Open `index.html` in your browser (works offline)
3. Log your daily training tasks, lock in your session, and watch your fighter grow

## Tabs

- **Daily** — Training tasks, round timer with custom duration, session log
- **Stats** — 5 attributes with radar chart and style archetype
- **Weight** — Weight log with sparkline trend chart
- **Ranks** — Progression syllabus with gatekeeper tests
- **Awards** — Achievements with unlock progress

## Save / Load

All data is stored in `localStorage`. Use the **Export / Import** buttons in the Ranks tab to back up or transfer your save.
